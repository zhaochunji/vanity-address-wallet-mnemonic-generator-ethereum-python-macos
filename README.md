# vanity-address-wallet-mnemonic-generator-ethereum-python-macos
macOS 平台高性能以太坊靓号地址与助记词钱包生成器 | High-performance Ethereum vanity address & mnemonic wallet generator for Python on macOS
---------------------------------------------------
## 🔥 项目功能
- 生成以太坊靓号地址（自定义前缀/后缀）
- 自动生成助记词（12/24词）
- 自动生成助记词、私钥、公钥、地址
- 纯本地离线运行，安全不联网
- 支持多进程加速生成
- 适配 macOS 系统
---------------------------------------------------
## 📌 环境要求
1. 电脑系统：**macOS**
2. 运行环境：**Python 3.8 及以上版本**
3. 网络：无需联网，纯本地运行
---------------------------------------------------
## 🚀 快速开始（一步一步教你）
### 1. 检查 Python 环境是否安装
打开电脑的 终端（Terminal） 输入以下命令：
python3 --version
如果显示版本号（例如 Python 3.9.6），说明环境正常。
如果没有安装 Python，请先去官网安装：
https://www.python.org/downloads/macos/

### 2. 下载eth.py源码文件到自己指定文件夹下
下载项目中eth.py文件到自己指定文件夹，.py文件的名字也可以自行修改，如果修改，步骤 ###4 需同步修改

### 3. 安装依赖库（必须执行，这一步需联网）
在eth.py文件所在的文件夹右键打开 终端（Terminal） 中输入：【pip3 install eth-account mnemonic】

### 4. 运行生成器
在终端中输入以下命令来启动生成器：【python3 eth.py】

📝 注意事项
确保使用 macOS 设备，并且已安装正确版本的 Python。
该工具在本地完全离线运行，不需要网络连接，确保更高的安全性。
如果在安装依赖或运行过程中遇到问题，检查 Python 和 pip 环境是否正确配置。

📋 参数设置说明
🎴 以太坊靓号地址生成器・配置参数详细说明
-----------------------------------------------------------------------------------------------
⚙️ 一、核心基础参数
ETH_ALPHA_CASE_SENSITIVE ✅ 字母检测是否区分大小写
True= 区分大小写 ｜ False= 不区分大小写
ETH_ADDR_OUTPUT_CASE 📤 导出地址大小写格式
original= 保留原大小写 ｜ upper= 全大写 ｜ lower= 全小写
-----------------------------------------------------------------------------------------------
🔢 二、顺子检测开关（核心规则）
ETH_SEQ_0_9_ENABLE / ETH_SEQ_0_9_MIN_LEN 🔁 0~9 循环正顺 + 最小匹配位数
ETH_SEQ_0_10_ENABLE / ETH_SEQ_0_10_MIN_LEN 📈 0~10 连贯正顺 + 最小匹配位数
ETH_REV_SEQ_0_9_ENABLE / ETH_REV_SEQ_9_MIN_LEN 🔂 0~9 倒序顺子 + 最小匹配位数
ETH_REV_SEQ_0_10_ENABLE / ETH_REV_SEQ_0_10_MIN_LEN 📉 0~10 连贯倒顺 + 最小匹配位数
-----------------------------------------------------------------------------------------------
🎯 三、基础连续相同字符阈值
ETH_START_DIGIT_SAME 🎰 开头连续相同数字阈值
ETH_START_ALPHA_SAME 🔤 开头连续相同字母阈值
ETH_END_DIGIT_SAME 🎰 结尾连续相同数字阈值
ETH_END_ALPHA_SAME 🔤 结尾连续相同字母阈值
ETH_MID_DIGIT_SAME 🎰 中间连续相同数字阈值
ETH_MID_ALPHA_SAME 🔤 中间连续相同字母阈值
-----------------------------------------------------------------------------------------------
🔁 四、字符重复检测配置
🧩 1 位字符重复检测
ETH_1CHAR_REPEAT_MUST_CONTINUOUS 📏 单字符重复是否要求连续出现
ETH_1DIGIT_REPEAT_CHECK + 阈值 🎲 单数字重复检测开关 + 最少出现次数
ETH_1ALPHA_REPEAT_CHECK + 阈值 🔡 单字母重复检测开关 + 最少出现次数
🧬 2–10 位数字 / 字母重复检测
ETH_REPEAT_MUST_CONTINUOUS 📌 多字符子串是否必须连续匹配
分别独立配置：2～10 位数字、2～10 位字母
每项可单独开关、自定义最小重复次数，不用可直接关闭
-----------------------------------------------------------------------------------------------
📝 五、自定义字符串检测
ETH_CUSTOM_STR_ENABLE 🧷 自定义靓号字符串总开关
ETH_CUSTOM_STR 📚 内置海量靓号库：豹子 / 顺子 / 叠字 / 生日 / 字母组合 / 对称格式等
ETH_CUSTOM_STR_CASE_SENSITIVE 🔠 自定义串是否区分大小写
ETH_CUSTOM_STR_START_ENABLE 🔝 匹配自定义串出现在地址开头
ETH_CUSTOM_STR_ANY_ENABLE 📍 匹配自定义串出现在地址任意位置
ETH_CUSTOM_STR_END_ENABLE 🔚 匹配自定义串出现在地址结尾
-----------------------------------------------------------------------------------------------
🛡️ 六、其他基础检测开关
ETH_CHECK_FULL_NUM 🀄 检测纯数字以太坊地址
ETH_CHECK_FULL_ALPHA 🔤 检测纯字母以太坊地址
ETH_CHECK_REPEAT_PATTERN 🔄 检测循环重复格式（如 121212、808080）
-----------------------------------------------------------------------------------------------
🚫 七、字符包含 / 排除检测
ETH_EXCLUDE_CHARS_ENABLE ❌ 开启排除指定字符功能
ETH_EXCLUDE_CHARS 📝 设置需要过滤掉的字符
ETH_ONLY_ALLOW_CHARS_ENABLE ✅ 开启仅允许指定字符模式
ETH_ONLY_ALLOW_CHARS_LIST 📋 限定地址只能包含的数字 / 字母组合池
-----------------------------------------------------------------------------------------------
🔝 八、前 10 位精细化规则配置（0x 后 1–10 位）
ETH_FIRST10_ENABLE 🎛️ 地址前 10 位规则总开关
ETH_FIRST2_ENABLE + 字符池 🎯 前 2 位匹配开关 + 自定义靓号字符池
ETH_FIRST2_CASE_SENSITIVE 🔡 前 2 位是否区分大小写
ETH_FIRST3_10_ENABLE 🧩 第 3–10 位子规则总开关
子规则分项
双炸弹匹配：开关 + 组合池子 + 大小写控制
顺子匹配：开关 + 顺子库 + 大小写控制
双字符叠词匹配：开关 + 组合池 + 大小写控制
回文对称匹配：开关 + 回文库 + 大小写控制
-----------------------------------------------------------------------------------------------
🔚 九、末尾 10 位精细化规则配置
ETH_LAST10_ENABLE 🎛️ 地址末尾 10 位规则总开关
ETH_LAST2_ENABLE + 字符池 🎯 最后 2 位匹配开关 + 靓号字符池
ETH_LAST2_CASE_SENSITIVE 🔡 最后 2 位是否区分大小写
倒数 3–10 位：直接复用前 10 位炸弹 / 顺子 / 双字符 / 回文池子，无需重复配置
-----------------------------------------------------------------------------------------------
📜 十、40 位全地址精细化字符规则
ETH_40CHAR_RULE_ENABLE 🧾 40 位完整地址逐位规则总开关
支持第 1～40 位每一位单独限定字符集
可分别设置：允许字符、是否区分大小写、自定义限制条件
适合极致精细化定制高端靓号筛选
-----------------------------------------------------------------------------------------------
🔂 十一、前缀 / 后缀重复模式配置
ETH_PREFIX_SUFFIX_REPEAT_ENABLE 🎚️ 前缀后缀重复功能总开关
📌 前缀重复（0x 开头部分）
支持 1～10 位 单元独立开关
可自定义每一种位数对应的最小重复次数
适配：123123123、88998899 这类循环前缀
📌 后缀重复（地址结尾部分）
同前缀逻辑，1～10 位可单独开启 / 关闭
每种位数自由设置重复次数门槛
ETH_PREFIX_SUFFIX_CASE_SENSITIVE 🔠 前缀后缀匹配是否区分大小写
-----------------------------------------------------------------------------------------------
💻 十二、程序运行基础参数
PROCESS_NUM ⚡ 运行 CPU 核心数，越大生成速度越快
PROGRESS_REFRESH ⏱️ 进度 / 速度刷新间隔（单位：秒）
SAVE_FILE_PATH 💾 靓号地址自动保存的本地文件路径
-----------------------------------------------------------------------------------------------

以上参数设置帮助你灵活地控制生成以太坊靓号地址的规则。通过开启或关闭不同的检测开关，你可以定制符合个人需求的地址生成器。
