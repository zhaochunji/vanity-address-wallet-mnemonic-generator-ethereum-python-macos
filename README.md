# vanity-address-wallet-mnemonic-generator-ethereum-python-macos
macOS 平台高性能以太坊靓号地址与助记词钱包生成器 | High-performance Ethereum vanity address & mnemonic wallet generator for Python on macOS
---------------------------------------------------
## 🔥 项目功能
- 生成以太坊靓号地址（自定义前缀/后缀）
- 自动生成助记词（12/24词）
- 自动生成助记词、私钥、公钥、地址
- 纯本地离线运行，安全不联网
- 支持多进程加速生成
- 适配 macOS 系统
---------------------------------------------------
## 📌 环境要求
1. 电脑系统：**macOS**
2. 运行环境：**Python 3.8 及以上版本**
3. 网络：无需联网，纯本地运行
---------------------------------------------------
## 🚀 快速开始（一步一步教你）
### 1. 检查 Python 环境是否安装
打开电脑的 终端（Terminal） 输入以下命令：
python3 --version
如果显示版本号（例如 Python 3.9.6），说明环境正常。
如果没有安装 Python，请先去官网安装：
https://www.python.org/downloads/macos/

### 2. 下载eth.py源码文件到自己指定文件夹下
下载项目中eth.py文件到自己指定文件夹，.py文件的名字也可以自行修改，如果修改，步骤 ###4 需同步修改

### 3. 安装依赖库（必须执行，这一步需联网）
在eth.py文件所在的文件夹右键打开 终端（Terminal） 中输入：【pip3 install eth-account mnemonic】

### 4. 运行生成器
在终端中输入以下命令来启动生成器：【python3 eth.py】

📝 注意事项
确保使用 macOS 设备，并且已安装正确版本的 Python。
该工具在本地完全离线运行，不需要网络连接，确保更高的安全性。
如果在安装依赖或运行过程中遇到问题，检查 Python 和 pip 环境是否正确配置。

📋 参数设置说明
【核心保留参数】
ETH_ALPHA_CASE_SENSITIVE
作用：决定地址字母是否区分大小写。
示例：如果设置为 True，地址中的字母 a 和 A 会被视为不同字符；设置为 False，则不区分大小写。
ETH_ADDR_OUTPUT_CASE
作用：控制生成的地址大小写。
示例：
"original"：保持原始大小写。
"upper"：全大写。
"lower"：全小写。
【顺子检测开关】
ETH_SEQ_0_9_ENABLE
作用：启用 0~9 顺序数字检测。
示例：如果开启，生成的地址中包含数字 0123456789 时会被识别为顺子。
ETH_SEQ_0_10_ENABLE
作用：启用 0~10 顺序数字检测。
示例：如果开启，地址中可以包含数字 01234567891011。
ETH_REV_SEQ_0_9_ENABLE
作用：启用 0~9 倒序数字检测。
示例：如果开启，地址中的数字 9876543210 会被识别为倒序顺子。
【重复检测】
ETH_1DIGIT_REPEAT_CHECK & ETH_1ALPHA_REPEAT_CHECK
作用：启用单一数字或字母重复检测。
示例：如果开启并设置 ETH_1DIGIT_REPEAT_MIN = 19，则检测地址中是否有数字或字母重复 19 次以上。
ETH_REPEAT_MUST_CONTINUOUS
作用：是否要求重复字符必须是连续的。
示例：如果设置为 True，则地址中的重复字符如 111111 会被识别为有效；如果设置为 False，则可以是非连续的重复字符。
【自定义字符串检测】
ETH_CUSTOM_STR_ENABLE
作用：启用自定义字符串检测。
示例：如果设置为 True，你可以指定一些自定义字符串（如 8888888），地址中包含这些字符串时会被视为靓号。
ETH_CUSTOM_STR
作用：指定要检测的自定义字符串。
示例：你可以设置字符串 8888888;9999999;123456，生成的地址包含这些字符串时，会被标记为靓号。
【字符包含/排除检测】
ETH_EXCLUDE_CHARS_ENABLE
作用：是否启用排除字符的检测。
示例：如果设置为 True，可以指定排除某些字符（如 4），生成的地址中将不允许出现这些字符。
ETH_ONLY_ALLOW_CHARS_ENABLE
作用：是否启用仅包含特定字符的检测。
示例：如果设置为 True，生成的地址必须只包含指定的字符组合，例如只包含数字和字母。
【前缀/后缀重复检测】
ETH_PREFIX_SUFFIX_REPEAT_ENABLE
作用：启用前缀/后缀重复模式检测。
示例：如果启用，你可以指定生成的地址中前缀或后缀是否重复，允许重复一定次数，如 123123123 或 67896789。
ETH_PREFIX_REPEAT_ENABLE & ETH_SUFFIX_REPEAT_ENABLE
作用：分别启用前缀和后缀的重复检测。
示例：如果设置为 True，地址中的前缀（例如 123123123123）或后缀（例如 67896789）可以重复一定次数。
【40位精细化规则】
ETH_40CHAR_RULE_ENABLE
作用：启用 40 位地址的精细化字符规则检测。
示例：如果开启，你可以指定更详细的规则，例如某些位置必须包含特定的字符，甚至可以指定偶数位和奇数位必须满足某些条件。
【运行参数】
PROCESS_NUM
作用：设置生成以太坊地址时的并发进程数。
示例：如果设置为 8，则会启动 8 个进程同时生成地址，提升生成速度。
SAVE_FILE_PATH
作用：设置生成的地址和助记词保存的文件路径。
示例：设置为 "save_address_eth.txt"，所有生成的地址和助记词将保存在该文件中。

以上参数设置帮助你灵活地控制生成以太坊靓号地址的规则。通过开启或关闭不同的检测开关，你可以定制符合个人需求的地址生成器。
